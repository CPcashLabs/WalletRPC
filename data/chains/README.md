# data/chains

## Directory Positioning
This directory defines supported chain presets for WalletRPC.

## Main Code Scope
- Network definitions for Ethereum, BSC, BTTC, TRON, and testnets.
- Unified chain export index for runtime consumption.

## Functional Role
- Provides canonical chain configs used by wallet initialization, explorer links, and token defaults.
- Enables extension of new chains by adding modular chain files.
