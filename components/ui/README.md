# components/ui

## Directory Positioning
This is the core reusable UI layer for WalletRPC.

## Main Code Scope
- Brand and identity components: `BrandLogo.tsx`.
- Generic interaction components: `Button.tsx`, `CountUp.tsx`, `TiltCard.tsx`, `ParticleIntro.tsx`.
## Functional Role
- Encapsulates base UI capabilities such as buttons, animated numbers, card effects, and intro visuals.
- Acts as a stable UI foundation for wallet onboarding, dashboard, and transaction flows.
