# 产品范围与原则

## 1. 产品范围（In Scope）
- 多链钱包基础能力：导入、地址展示、余额与代币资产读取。
- 交易能力：原生转账、代币转账、交易状态追踪。
- 多签能力：Safe 跟踪、提议、签名、执行、成员治理。
- 网络能力：链配置管理、RPC 配置、区块浏览器跳转。
- 本地持久化：用户自定义网络、代币、Safe 跟踪信息。

## 2. 非范围（Out of Scope）
- 托管私钥与服务端代持。
- 复杂投资功能（理财、收益策略、自动化交易）。
- 跨链桥聚合与路由优化。
- 中心化账户体系（手机号/邮箱注册登录）。

## 3. 核心原则
- 安全优先：敏感信息最小驻留，失败默认安全。
- 可解释优先：用户看到的状态必须有明确意义。
- 一致性优先：同一业务概念在文档、代码、测试中定义一致。
- 低冗余请求：无明确价值的自动刷新应被禁止。
