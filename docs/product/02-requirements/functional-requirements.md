# 功能需求（FR）

## FR-001 钱包导入
- 描述：支持助记词和私钥导入，生成 EVM 与 TRON 地址信息。
- 验收：导入成功后能进入主界面并展示地址。

## FR-002 资产展示
- 描述：展示原生资产与代币余额，支持手动刷新。
- 验收：余额与代币列表可见，刷新按钮可触发更新。

## FR-003 交易发送
- 描述：支持原生币与代币转账，包含输入校验与状态反馈。
- 验收：非法参数被拦截；合法请求进入发送流程并返回结果。

## FR-004 交易状态追踪
- 描述：对已提交交易进行状态确认，直至成功/失败/超时。
- 验收：状态变更可见；超时应终止无意义轮询并给出可解释结果。

## FR-005 Safe 多签流程
- 描述：支持 Safe 跟踪、提议、签名、执行、成员管理。
- 验收：可在队列页看到签名进度并执行满足阈值的提案。

## FR-006 网络与代币配置
- 描述：支持网络参数维护、代币导入/编辑/删除。
- 验收：配置变更可持久化，刷新页面后仍生效。

## FR-007 日志与可追踪性
- 描述：展示交易记录并可跳转区块浏览器。
- 验收：每条有 hash 的记录可外链查看。
