# 产品目标

## 1. 产品使命
提供一个可控、可审计、可扩展的 Wallet RPC 客户端，让用户在 EVM 与 TRON 网络中，以统一体验完成资产管理与交易执行。

## 2. 北极星目标
- 让用户在最少操作下完成“导入 -> 查看资产 -> 发起交易 -> 跟踪结果”的完整闭环。

## 3. 阶段性目标（当前）
- 稳定：核心交易链路成功率和可恢复性可接受。
- 清晰：关键状态（待确认、成功、失败）可解释。
- 高效：避免无意义重复请求，控制 RPC 消耗。
- 可维护：产品逻辑以文档驱动，减少沟通摩擦。

## 4. 成功指标（建议）
- 首次导入到可见余额耗时。
- 交易状态追踪成功率。
- 每会话平均冗余请求数。
- 需求变更从文档到上线的周期。
